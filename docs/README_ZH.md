MyWallet 是一款现代化的个人财务追踪 Android 应用，完全使用 Kotlin 构建。它采用“离线优先”策略，使用本地 Room 数据库确保您的数据快速、私密且始终可用，同时提供智能财务分析和便捷的主屏幕小部件。

---

## ✨ 功能特性

### 📱 核心应用功能

* 📊 **实时仪表盘:** 查看您的当前余额、总收入和总支出，均可按时间（所有时间、本月、今年）筛选。
* 📈 **可视化图表:**
    * **甜甜圈图** 让您直观地看到“钱花在哪儿了”。
    * **柱状图** 用于比较您的总收入与总支出。
* ✍️ **记录所有:** 轻松记录收入和支出，包含日期、描述、金额和类别。
* ⭐ **智能愿望清单:**
    * 添加您想购买的物品。
    * 立即根据您的余额进行**购买力检查**。
    * 根据您的储蓄习惯查看**储蓄时间线**（例如，“预计 5 个月”）。
* ✏️ **编辑与删除:**
    * 点击任何交易上的设置图标可对其进行编辑或删除。
    * 点击愿望清单项目上的设置图标可对其进行编辑。
    * 将愿望清单项目标记为“已完成”，编辑图标会巧妙地变为删除图标。
* 🔔 **通知:** 每次成功添加新交易时，立即获得本地通知。
* ⏰ **每日提醒:** 设置自定义的每日提醒（默认晚上 9:00），让您永远不会忘记记录财务。
* 🎨 **主题设置:** 立即在**浅色**、**深色**或**跟随系统**主题之间切换。
* 🔒 **离线与私密:** 所有数据都安全地存储在您设备上的私有 Room 数据库中。无需帐户或互联网连接。

### 🏠 主屏幕小部件

* 💰 **统计小部件:** 一目了然地查看您的余额、收入和支出总额。当您在应用中进行更改时，它会立即更新。
* 🛒 **愿望清单小部件:** 一个可滚动的交互式列表。您可以直接在主屏幕上勾选您的目标。
* ➕ **快速添加小部件:** 一个简单的按钮，可打开专用屏幕，无需打开完整应用即可快速添加新交易。

## 🛠 技术栈

* **语言:** 100% **Kotlin**
* **架构:** **MVVM** (ViewModel, LiveData)
* **数据库:** **Room** (用于本地设备上的 SQL 数据库存储)
* **UI:** **ViewBinding**, **Android Navigation Component**, Material Components
* **并发:** **Kotlin Coroutines**
* **异步:** `AlarmManager` & `BroadcastReceiver` (用于每日提醒)
* **图表:** **MPAndroidChart**
* **小部件:** **AppWidgets** (使用 `RemoteViews` & `ListView`)

---

## 🚀 如何构建

本项目使用 Gradle wrapper 构建。

1.  克隆此仓库。
2.  连接 Android 设备或启动模拟器。
3.  从项目根目录运行 `gradlew installDebug`。
