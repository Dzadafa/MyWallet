**MyWallet** は、**Kotlin** で完全に構築された、モダンな個人用ファイナンス管理アプリです。  
安全な **Firebase** バックエンドを利用し、リアルタイムデータの同期、スマートな財務分析、そして便利なホーム画面ウィジェットを提供します。

このアプリは、あなたの「お金の健康状態」を直感的に把握し、支出を管理し、貯蓄目標の達成をサポートするために設計されています。

---

## ✨ 機能概要

### コアアプリ機能
* **収支記録：** 日付、説明、金額（ルピア）、カテゴリを入力して収入や支出を記録。  
* **取引履歴：** 「最近の収入」「最近の支出」に分類された過去の取引を閲覧可能。  
* **リアルタイムダッシュボード：** 現在の残高（総収入－総支出）を表示。残高がマイナスの場合は赤色で警告表示。  
* **期間フィルター：** 「すべての期間」「今月」「今年」ごとのデータを表示。  
* **ビジュアルチャート：**
    * **支出内訳チャート：** 支出をカテゴリごとにまとめたドーナツチャート。  
    * **収入 vs 支出チャート：** 指定期間内の収入と支出の合計を比較する棒グラフ。  
* **スマートウィッシュリスト：**
    * 欲しいアイテムと価格を登録。  
    * **購入可能チェック：** 現在の残高と比較し、購入可能かどうかを即時表示。  
    * **貯蓄期間の予測：** 現在の貯蓄ペースをもとに、「約 5 か月後に購入可能です」などの目安を提示。  
    * **チェックリスト機能：** 購入済みアイテムを「完了」としてマークし、取り消し線付きでリスト下部に移動。  
* **安全なクラウド保存：** すべてのデータは Firebase Firestore に安全に保存され、初回起動時に匿名ユーザーアカウントが自動生成されます。  
* **テーマ切り替え：** **ライトモード**、**ダークモード**、または **システム設定に従う** を即座に切り替え可能。
