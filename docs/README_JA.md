MyWalletは、Kotlinのみで構築された、Android向けのモダンな個人財務管理アプリです。オフラインファースト設計を採用し、ローカルのRoomデータベースを使用して、データを高速かつプライベートに保ち、常に利用可能にします。同時に、スマートな財務分析と便利なホーム画面ウィジェットを提供します。

---

## ✨ 機能概要

### 📱 アプリのコア機能

* 📊 **リアルタイムダッシュボード:** 現在の残高、総収入、総支出を表示し、すべて時間（全期間、今月、今年）で絞り込み可能です。
* 📈 **ビジュアルチャート:**
    * **ドーナツチャート** で「お金がどこに使われているか」を視覚化。
    * **棒グラフ** で総収入と総支出を比較。
* ✍️ **すべてを記録:** 日付、説明、金額、カテゴリを含む収支を簡単に記録。
* ⭐ **スマートウィッシュリスト:**
    * 購入したいアイテムを追加。
    * 残高と照らし合わせた**購入可能か**を即座にチェック。
    * あなたの貯蓄ペースに基づいた**貯蓄タイムライン**（例：「約5ヶ月」）を表示。
* ✏️ **編集と削除:**
    * 各取引の設​​定アイコンをタップして、編集または削除。
    * ウィッシュリスト項目の設​​定アイコンをタップして編集。
    * ウィッシュリスト項目を「完了」にすると、編集アイコンが削除アイコンに変わります。
* 🔔 **通知:** 新しい取引を追加するたびに、即座にローカル通知を受け取れます。
* ⏰ **デイリーリマインダー:** 毎日のリマインダーをカスタム時間（デフォルトは午後9時）に設定し、家計簿の付け忘れを防ぎます。
* 🎨 **テーマ設定:** **ライト**、**ダーク**、または**システムデフォルト**のテーマに即座に切り替え。
* 🔒 **オフライン＆プライベート:** すべてのデータはデバイス上のプライベートなRoomデータベースに安全に保存されます。アカウントやインターネット接続は不要です。

### 🏠 ホーム画面ウィジェット

* 💰 **統計ウィジェット:** 残高、収入、支出の合計が一目でわかります。アプリでの変更は即座に反映されます。
* 🛒 **ウィッシュリストウィジェット:** スクロール可能でインタラクティブなリスト。ホーム画面から直接目標をチェックできます。
* ➕ **クイック追加ウィジェット:** 専用画面を開くシンプルなボタン。アプリ全体を開かずに、新しい取引をすばやく追加できます。

## 🛠 技術スタック

* **言語:** 100% **Kotlin**
* **アーキテクチャ:** **MVVM** (ViewModel, LiveData)
* **データベース:** **Room** (ローカルデバイス上のSQLデータベースストレージ)
* **UI:** **ViewBinding**, **Android Navigation Component**, Material Components
* **並行処理:** **Kotlin Coroutines**
* **非同期処理:** `AlarmManager` & `BroadcastReceiver` (デイリーリマインダー用)
* **グラフ:** **MPAndroidChart**
* **ウィジェット:** **AppWidgets** (`RemoteViews` & `ListView`を使用)

## 🚀 ビルド方法

このプロジェクトはGradle wrapperを使用します。

1.  リポジトリをクローンします。
2.  Androidデバイスを接続するか、エミュレータを起動します。
3.  プロジェクトのルートディレクトリから `gradlew installDebug` を実行します。
